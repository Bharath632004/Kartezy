# Operations Runbooks

Standard operating procedures for the Kartezy platform.

## Contents

| Runbook | Description |
|---------|-------------|
| `service-restart.md` | Graceful service restart procedure |
| `deployment-rollback.md` | Rolling back a failed deployment |
| `database-recovery.md` | Database backup restoration |
| `incident-response.md` | Severity classification and response protocols |
| `kafka-recovery.md` | Kafka cluster recovery and rebalancing |
| `redis-eviction.md` | Cache eviction and warming strategies |
| `ssl-renewal.md` | SSL certificate renewal process |
| `scaling-playbook.md` | Horizontal pod autoscaling adjustments |

## Principles
- All runbooks follow the "check → decide → act → verify" pattern
- Commands are provided as copy-paste ready snippets
- Expected outcomes and validation steps are documented
