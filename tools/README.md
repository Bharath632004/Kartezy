# Kartezy Developer Tools

Internal tooling and utilities for Kartezy engineering teams.

## Planned Tools

- **Service Scaffolder** — CLI to generate new microservice scaffolding
- **Database Migrator** — Tool to manage and run database migrations across environments
- **API Client Generator** — Auto-generate API clients from OpenAPI specs
- **Flutter Code Generator** — Generate feature modules, models, providers
- **Environment Manager** — Switch between dev/staging/prod configs
- **Secrets Manager** — Encrypt/decrypt secrets for different environments
