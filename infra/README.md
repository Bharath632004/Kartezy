# Infrastructure

This directory contains infrastructure scripts and configurations for the Kartezy platform.

## Structure

```
infra/
├── scripts/         # Infrastructure automation scripts
└── README.md
```

## Contents

- **Terraform**: For cloud resource provisioning (see `devops/terraform/` if available)
- **Helm Charts**: For Kubernetes package management
- **Monitoring**: Prometheus/Grafana configurations
- **Backup Scripts**: Database and service backup automation

## Usage

Refer to the [DevOps Guide](../devops/README.md) for detailed infrastructure deployment instructions.
