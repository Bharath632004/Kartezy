# Kartezy Shared Libraries

Cross-platform shared libraries used by all Kartezy applications (Flutter mobile apps, web apps, and services).

## Libraries

| Library | Description | Status |
|---------|-------------|--------|
| `lib/networking/` | HTTP client, API interceptors, retry logic | 📋 Planned |
| `lib/auth/` | Authentication helpers, token management | 📋 Planned |
| `lib/analytics/` | Event tracking, analytics integration | 📋 Planned |
| `lib/theme/` | Design tokens, theming constants | 📋 Planned |
| `lib/localization/` | i18n, translations, locale helpers | 📋 Planned |
| `lib/utils/` | Common utility functions | 📋 Planned |
| `lib/storage/` | Local storage abstraction layer | 📋 Planned |
| `lib/logging/` | Structured logging | 📋 Planned |
| `lib/widgets/` | Reusable UI components | 📋 Planned |
| `lib/maps/` | Map integration utilities | 📋 Planned |
| `lib/payments/` | Payment gateway integration helpers | 📋 Planned |
| `lib/validation/` | Input validation rules | 📋 Planned |
| `lib/security/` | Encryption, secure storage | 📋 Planned |
| `lib/notifications/` | Push notification helpers | 📋 Planned |
| `lib/constants/` | App-wide constants and enums | 📋 Planned |
